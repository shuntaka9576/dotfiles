# Project Structure

## Root Directory
```
dotfiles/
├── flake.nix              # Nix flake definition
├── flake.lock             # Dependency lock file
├── Makefile               # Build and maintenance commands
├── install.sh             # Initial installation script
├── treefmt.nix            # Formatter configuration
├── stylua.toml            # Lua formatter configuration
├── renovate.json          # Automatic dependency update configuration
├── nvfetcher.toml         # Nix package source update configuration
├── README.md              # Project documentation
├── .gitignore             # Git exclusion settings
└── .rgignore              # ripgrep exclusion settings

## Main Directories
### /home-manager/
- `home.nix` - Main Home Manager configuration
- `/programs/` - Individual configurations for each program
  - Each program has its own directory
  - Each directory contains a `default.nix`

### /nix-darwin/
- `default.nix` - macOS system configuration

### /overlays/
- Nix package customizations

### /_sources/
- Package sources generated by nvfetcher (auto-generated)

### /.serena/
- Serena MCP server working directory

## Important Configuration Files
- `/home-manager/programs/mcp/` - MCP (Model Context Protocol) configuration
  - `secrets.jsonnet.example` - Secrets template
  - `mcp-general.jsonnet` - General MCP configuration
  - `mcp-claude-code.jsonnet` - Claude Code MCP configuration

- `/home-manager/programs/claude/` - Claude AI integration
  - `/hooks/` - File edit hooks (format.ts, notify.ts)
